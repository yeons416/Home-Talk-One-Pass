주차 이용 안내

1. 입주자 차량 등록

등록 방법

  1. 주차 메뉴 → 입주자 차량 조회 → 차량 추가 버튼 클릭
  2. 차량 번호, 차량 모델, 차량 종류 입력
  3. 자동차 등록증 사진/파일 첨부 후 제출

필요 서류

  • 자동차 등록증 (사진 또는 PDF)
  • 관리사무소에 직접 제출도 가능합니다.

승인 절차

  1. 서류 제출 후 관리자 검토 (승인 대기 상태)
  2. 승인 완료 시 입출차 및 알림 서비스 제공
  3. 반려 시 반려 사유 확인 후 재신청 가능

2. 방문 차량 예약

예약 방법

  1. 주차 메뉴 → 사전 차량 예약 → 예약하기 버튼 클릭
  2. 차량 번호, 방문 목적, 방문 예정 시각 입력 후 등록
  3. 방문 차량 입차 시 자동으로 세대 정보가 연결됩니다.

취소 방법

  1. 주차 메뉴 → 사전 차량 예약에서 예약 목록 확인
  2. 취소할 예약의 취소 버튼 클릭
  3. 입차 완료된 예약은 취소할 수 없습니다.

3. 티켓 사용 방법

티켓 종류

  • 1시간권 — 1시간 무료 주차 제공
  • 1일권 — 24시간 무료 주차 제공

적용 방법

  1. 주차 메뉴 → 티켓 등록에서 차량 번호 4자리 입력 후 조회
  2. 적용할 티켓 종류 선택 후 수량 조절
  3. 적용 버튼 클릭
  4. 출차 전까지 티켓 적용 취소 및 수정 가능

발급 주기

  • 매월 1일 세대별 자동 발급
  • 미사용 티켓은 다음 달로 이월되지 않습니다.

4. 주차 기록 조회

조회 방법

  1. 주차 메뉴 → 차량 조회 클릭
  2. 년/월 선택으로 원하는 달의 주차 기록 조회
  3. 입주자 차량 및 방문 차량의 입출차 기록을 확인할 수 있습니다.

5. 전기차 충전소 위치

전기차 충전소 위치 정보는 관리사무소에 문의해주세요.
